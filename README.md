## 사용 방법
1. 다운로드를 받는다.
2. 압축을 해제 후 편집기(vsc, notepad++, 메모장 등)로 열어준다.
3. 열었으면 main.js의 30번째 줄으로 가서 'Your discord channel webhook url'이라고 써있는 곳에 웹훅URL를 입력
4. 입력이 끝났으면 index.html을 실행한다.(vsc인 경우 "Live server"를 이용하여 열어주면 로컬호스트로 열 수 있다.)
